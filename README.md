## 🔬 About Me
I am a PhD researcher working on human gait analysis using wearable sensors.  
My research focuses on detecting gait events and predicting push-off timing using deep learning models and IMU data.

My goal is to support the development of assistive technologies such as exoskeletons and prosthetic devices by providing accurate and real-time gait phase detection.

## 🚀 Current Projects

### 1. IMU-Based Gait Event Detection
- Developed a deep learning model to detect:
  - Heel Strike (HS)
  - Foot Flat (FF)
  - Toe Off (TO)
  - Push-Off Phase  
- Sensors: Shank & Foot IMUs  
- Goal: Improve timing for assistive devices  

---

### 2. Push-Off Timing Prediction
- Focus on identifying the most critical phase in gait  
- Enables:
  - Better propulsion assistance  
  - Improved gait symmetry  
  - Reduced energy cost
