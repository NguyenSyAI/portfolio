# Projects

## 🤖 IMU-Based Gait Event Detection

### 📌 Description
This project develops a deep learning model to detect gait events using only two IMU sensors (shank and foot).

### 🎯 Detected Events
- Heel Strike (HS)  
- Foot Flat (FF)  
- Toe Off (TO)  
- Push-Off Phase

## 🧠 Methods
- Signal filtering and preprocessing  
- Deep learning (CNN, LSTM)  
- Time-series classification  
- Imbalanced learning (Macro F1-score)  

---

## 🚀 Impact
- Improves push-off timing detection  
- Supports assistive devices (exoskeletons, prosthetics)  
- Enhances gait symmetry and efficiency
