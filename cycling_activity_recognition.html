Attention‑Based BiLSTM for Cycling Activity Recognition Using Smartphones

IEEE Access, 2023 — DOI: 10.1109/ACCESS.2023.3338137

📌 Title

Attention Mechanism‑Based Bidirectional Long Short‑Term Memory for Cycling Activity Recognition Using Smartphones

👥 Authors

Van Sy Nguyen, Hyunseok Kim, and Dongjun Suh

🔎 Short Abstract / Overview

This work proposes a 1D‑CNN + BiLSTM model enhanced with an Attention Mechanism (CBiAM) for recognizing cyclists’ states using only a smartphone placed in the user’s pocket. The study also introduces the Cycling Safe (CySa) dataset, containing five cycling‑related actions: standing, sitting, walking with a bicycle, cycling, and falling. The proposed model achieves 100% F1‑score on the CySa dataset and demonstrates strong robustness across public HAR datasets including UCI‑HAR, PAMAP2, Opportunity, MOTIONSENSE, and WISDM.

🎯 Motivation

Cycling is increasingly popular but comes with significant safety risks, including bicycle‑human collisions and fall‑related injuries. Smartphones already contain rich inertial sensors, making them ideal for real‑time monitoring of cyclists’ conditions. This work aims to build a smartphone‑based safety system capable of detecting dangerous events (e.g., falls) and sharing the cyclist’s status with nearby riders for rapid assistance.

🚀 Main Contributions

  • Introduced the CySa dataset with five cycling‑related activities collected from smartphone sensors.
  • Developed a CBiAM architecture combining 1D‑CNN, BiLSTM, and Attention Mechanism.
  • Performed extensive hyperparameter studies (learning rate, batch size, sliding window ratios).
  • Achieved 100% F1‑score on the CySa dataset.
  • Demonstrated strong generalization on five public HAR datasets.

🧠 Method

The proposed CBiAM model consists of:
  • 1D‑CNN for spatial feature extraction
  • BiLSTM for temporal modeling in forward and backward directions
  • Attention Mechanism to weight important time steps
  • Softmax classifier for five activity classes
The attention layer computes relevance weights using:
Z(Q, K, V) = Softmax(QKᵀ / √dm) V

📊 Dataset / Experiments

The CySa dataset was collected using an iPhone 8 at 10 Hz. Five participants performed five actions for ~3 minutes each:
  • Standing (10,902 samples)
  • Sitting (11,472 samples)
  • Walking with bicycle (10,256 samples)
  • Cycling (11,334 samples)
  • Falling (10,789 samples)
Features extracted: - Mean, median, variance, standard deviation - Normalized using min‑max scaling

📈 Results

CySa Dataset

The proposed CBiAM model achieved:
  • F1‑score: 100%
  • Outperformed SVM, ANN, 1D‑CNN, LSTM, BiLSTM, and CNN‑LSTM

Public HAR Datasets

Weighted F1‑scores:
  • UCI‑HAR: 97.51%
  • PAMAP2: 99.82%
  • Opportunity LLL: 94.72%
  • WISDM: 97.67%
  • MOTIONSENSE: 87.05%
