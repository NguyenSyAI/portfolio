DeepCNN-Swin Transformer-Based Gait Cycle Event Prediction

A hybrid convolutional and Swin Transformer model for detecting conventional and push-off-related gait events across variable walking speeds using only shank and foot IMUs.

Overview

Accurate gait event prediction is essential for adaptive control of lower-limb assistive devices, including prostheses, ankle-foot orthoses, and exoskeletons. Many existing methods focus on common events such as heel strike, foot flat, and toe off, but they often ignore push-off-related transitions that are critical for timing energy release and propulsion assistance.

This project proposes a DeepCNN-Swin Transformer framework that detects both conventional gait events and push-off-related events across slow, normal, and fast treadmill walking speeds.

Main Contributions

  • Detects HS, MAD, FF, HR, MAU, and TO using only two wearable IMUs.
  • Introduces push-off-related event prediction to support adaptive prosthesis and AFO control.
  • Combines local temporal feature extraction from CNNs with broader temporal modeling from Swin Transformer blocks.
  • Outperforms SVM, KNN, DT, LDA, MLP, DNN, and LSTM-DNN baselines.

Detected Events

The model predicts six gait cycle events. MAD and MAU are included because they provide useful timing information for push-off assistance, where assistive devices may need to store and release energy at the right phase.

  • HS: Heel Strike
  • MAD: Maximum Acceleration Down
  • FF: Foot Flat
  • HR: Heel Rise
  • MAU: Maximum Acceleration Up
  • TO: Toe Off

Dataset and Features

Ten participants were recruited, with one participant omitted because of data errors. Each participant walked on a treadmill for three minutes at slow, normal, and fast speeds. Six IMUs were placed on the body during data collection, but the model uses only two IMUs from one leg: shank and foot.

Sixteen input features were extracted from acceleration and gyroscope signals, including three-axis acceleration, acceleration norm, three-axis angular velocity, and gyroscope norm for both shank and foot IMUs.

Pipeline

  1. Collect treadmill walking data.
    Participants walked at slow, normal, and fast speeds while wearable IMU data were recorded.
  2. Extract IMU features.
    Acceleration, gyroscope, and norm-based features were generated from shank and foot IMUs.
  3. Generate event labels.
    Conventional events and push-off-related events were annotated, with thick labeling applied around target events.
  4. Train DeepCNN-Swin Transformer.
    A CNN stem and residual blocks extract local patterns, while Swin Transformer blocks model longer temporal dependencies.
  5. Evaluate across subjects and speeds.
    Five-fold cross-validation was used to test generalization across unseen subjects.

Performance Comparison

The proposed DeepCNN-Swin Transformer achieved the highest overall performance among the compared methods, with a precision of 0.949 ± 0.015, recall of 0.954 ± 0.007, and F1-score of 0.951 ± 0.011.

Key Findings

  • Learning rate strongly affected convergence; 10-3 produced the most stable training and validation behavior.
  • HS and TO were detected with very high accuracy because their signal patterns are more distinct.
  • MAU achieved strong detection performance, which is important for timing push-off assistance.
  • HR was more challenging because its kinematic pattern is closer to adjacent non-event regions.

Application

The proposed model can support real-time gait phase prediction for active lower-limb assistive devices. By detecting push-off-related events, the model provides timing information that can help prostheses, AFOs, and exoskeletons activate or release assistance more naturally during walking.
